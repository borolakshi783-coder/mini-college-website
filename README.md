# Mini College Website

## Project Description

A simple and responsive college website developed using HTML5 and CSS3. The website provides basic information about the college, courses, and contact details with a clean and user-friendly interface.

## Features

- Responsive homepage
- About page
- Courses page
- Contact page
- Simple navigation menu
- Clean and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- Git & GitHub
